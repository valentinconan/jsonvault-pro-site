This repository serves two purposes:

1. **Public website** hosted via GitHub Pages — landing page and privacy policy for JSONVault Pro.
2. **Issue tracker** — bug reports and feature requests from users.

---

## JSONVault Pro

A Chrome extension (MV3) that transforms raw JSON, YAML and XML responses into a clean, interactive, searchable interface — directly in your browser.

- **Free tier:** tree view, syntax highlighting, search, multi-format support (JSON/YAML/XML), diff tool, DevTools panel, standalone viewer.
- **Pro tier:** JWT decode, Base64 detection, JSON Schema validation, JSONPath queries, export (YAML/XML/CSV/TypeScript), document history, virtual tree for large files, 21 premium themes, live editing.

## Reporting a bug

Please use the [Bug Report](.github/ISSUE_TEMPLATE/bug_report.md) template and include:

- Chrome version and OS
- Extension version (visible in `chrome://extensions`)
- Steps to reproduce
- Expected vs actual behavior
- A sample document if relevant (anonymize sensitive data)
